WPF에서 Win32 HWND 호스팅 샘플

[This readme is a partial copy of the sample's topic in the SDK documentation.]

이 샘플에서는 hwnd를 WPF(Windows Presentation Foundation) 응용 프로그램 안에 넣는 방법을 보여 줍니다.

이 샘플에서는 WPF(Windows Presentation Foundation)의 특정 기능에 대해 설명하므로 최선의 응용 프로그램 개발 방법을 따르지 않습니다. WPF(Windows Presentation Foundation) 및 Microsoft .NET Framework 응용 프로그램 개발을 위한 최선의 방법에 대한 자세한 내용은 다음에서 적절한 항목을 참조하십시오.

내게 필요한 옵션 - 액세스 가능성에 대한 유용한 정보

보안 - Windows Presentation Foundation 보안

지역화 - WPF 전역화 및 지역화 개요

샘플 빌드

  • Windows SDK(소프트웨어 개발 키트)를 설치하고 빌드 환경 명령 창을 엽니다. 시작 메뉴에서 모든 프로그램, Microsoft Windows SDK를 차례로 가리킨 다음 CMD 셸을 클릭합니다.

  • SDK(소프트웨어 개발 키트) 설명서 등에서 샘플을 하드 디스크 드라이브로 다운로드합니다.

  • 빌드 환경 명령 창에서 샘플을 빌드하려면 샘플의 소스 디렉터리로 이동합니다. 명령 프롬프트에 MSBUILD를 입력합니다. 두 하위 디렉터리의 프로젝트 파일을 참조하는 솔루션 파일을 빌드합니다.

  • Microsoft Visual Studio에서 샘플을 빌드하려면 샘플 솔루션이나 프로젝트 파일을 로드한 다음 Ctrl+Shift+B를 누릅니다.WPFApplication1.exe 파일을 디버그 세션 도중 실행될 실행 파일로 지정해야 하며 이 작업은 프로젝트 설정 대화 상자에서 수행합니다.

샘플 실행

  • 빌드 환경 명령 창에서 샘플을 실행하려면 WPFApplication1 소스 하위 디렉터리에 포함된 Bin\Debug 폴더의 WPFApplication1.exe 파일을 실행하십시오.

  • Visual Studio에서 디버깅 기능을 사용하여 컴파일된 샘플을 실행하려면 F5 키를 누릅니다.WPFApplication1.exe 파일을 디버그 세션 도중 실행될 실행 파일로 지정해야 하며 이 작업은 프로젝트 설정 대화 상자에서 수행합니다.
